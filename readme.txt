Simple Contact Button: Add a customizable contact button to your website, allowing visitors to connect with you instantly and easily.

== Description ==

Simple Contact Button is a powerful and easy-to-use WordPress plugin that enhances user engagement and communication on your website by adding a customizable contact button. This plugin allows businesses and individuals to offer visitors an immediate and seamless way to connect through platforms like WhatsApp. By making communication effortless, you can boost user interaction and improve conversion rates, whether you're running an e-commerce site, blog, or corporate website. The plugin's flexibility allows you to customize the button's appearance, position, and visibility to match your website's design, ensuring an optimal user experience across different devices. With its minimal setup, you can quickly integrate a professional and responsive contact option that positions your brand as approachable and customer-focused, helping to build stronger relationships with your audience.

We've introduced a new Plus plan to enhance your experience with additional premium features.

Our Plus plan includes advanced features such as displaying the contact button on specific pages, posts, or only on the homepage. Additionally, an analytics integration has been added to track the number of clicks on the button. Upgrade to Plus to access these exclusive features!



**Key Features:**
* Easy Setup: Quickly add a contact button to your website with minimal configuration.
* Customizable Position: Position the button at the bottom-left or bottom-right corner.
* Device Visibility Control: Display the button on desktop, mobile, or both.
* Initial Message: Customize the message that users see when they start a chat.
* Multiple Icon Choices: Choose from various icons to match your website's design.
* Page-Specific Visibility (Plus Plan): Display the contact button on specific pages, posts, or only on the homepage.
* Click Tracking (Plus Plan): Integrated analytics to track the number of clicks on the contact button.

== Installation ==

1. **Manual Installation:**
    * Upload the `simple-contact-button` folder to the `/wp-content/plugins/` directory using an FTP client.
    * Activate the plugin through the 'Plugins' screen in WordPress.
    * Navigate to Settings > Simple Contact Icon to configure the plugin settings.

2. **Installation via WordPress Dashboard:**
    * Go to Plugins > Add New, search for "Simple Contact Button", and click "Install Now".
    * After installation, click "Activate".
    * Go to Settings > Simple Contact Icon to configure your desired settings.

== Frequently Asked Questions ==

= How can I customize the position of the contact button? =

You can customize the button's position directly from the plugin's settings page. Choose to position it at the bottom-left or bottom-right corner.

= Is it possible to control which devices display the contact button? =

Yes, the plugin offers device visibility options. You can select to show the button on desktop, mobile devices, or both.

= How do I set a custom initial message? =

In the plugin settings under "Initial Message," you can set a pre-filled message that users will see when they start a chat.

= Can I use custom icons for the contact button? =

Yes, the plugin allows you to choose from multiple icons.

= Can I track button click data? =
Yes, with our Plus plan, button clicks are tracked and sent to GA4 as events.

= How to track button click data via GA4? =
When you upgrade to our Plus plan and integrate GA4, event tracking is automatically enabled. No additional setup is required. Use the following parameters to monitor the data:
* Event Name: contact_button_click
* Event Parameter: ep.event_category = Contact
This data will help you track user interactions effectively.
